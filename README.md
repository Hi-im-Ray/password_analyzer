# 🔐 Password Strength Analyzer Pro

> Cyberpunk-themed Password Security Toolkit for Python

A professional terminal-based password analysis and generation tool built with Python.  
Designed with a futuristic cyberpunk CLI interface using `Rich`, `Colorama`, and `Pyfiglet`.

---

## ✨ Features

### 🔎 Password Analysis
- Detect weak passwords
- Entropy calculation
- Brute-force crack time estimation
- Sequential & keyboard pattern detection
- Common password database checking
- Name & birthday pattern detection
- Character variety analysis

### 🔐 Secure Password Generation
Generate highly secure passwords in multiple styles:
- Gaming
- Hacker
- Professional
- Random

**Supports:**
- Leetspeak transformations
- Custom length
- Strong symbol combinations
- Multi-password generation

### 📁 Report System
- Export analysis reports
- Save generated passwords
- Timestamped TXT reports

### 🎨 Cyberpunk CLI Interface
- Animated loading screens
- Neon-themed UI
- Rich tables & panels
- ASCII-art banners
- Strength meters
- Smooth terminal experience

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Hi-im-Ray/password_analyzer.git
cd password_analyzer
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

Or manually:

```bash
pip install rich colorama pyfiglet
```

---

🚀 Launch

```bash
python3 password_analyzer.py
```
